📊 Masukkan Spreadsheet ID

ID otomatis tertanam di kode GAS

⚠️ Penting! Spreadsheet ID langsung masuk ke kode GAS. Tinggal copy-paste!

📌 Cara Mendapatkan Spreadsheet ID
  1. Buka Google Spreadsheet → buat baru
  2. Lihat URL: https://docs.google.com/spreadsheets/d/ID_ANDA/edit
  3. Copy bagian merah → paste di atas
2

📝 Copy Kode GAS (ID Tertanam)

Kode sudah berisi Spreadsheet ID — tinggal copy & paste

// Di dalam kode GAS:

const SPREADSHEET_ID = 'BELUM_DIISI';


            

                
📋 Cara Paste ke Apps Script:

                
    
                    
  1. Buka Spreadsheet → Extensions → Apps Script
    2. 
                    
  2. Hapus kode default
    3. 
                    
  3. Klik "📋 COPY KODE GAS" di atas → Paste (Ctrl+V)
    4. 
                    
  4. Klik 💾 Save (Ctrl+S)
    5.
